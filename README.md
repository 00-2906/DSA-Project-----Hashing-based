# DSA-Project-----Hashing-based
Login system made by using data structure named Hashing using the method of liner chaining.
This system is console based and it provides variety of functionality through it's various function.
Hashing is a data structure which creates hash valies for the easy storage and retrewal of data and liner chaining is used to create a link of the data using linked lists.
Functions it includes are:
1. Register User
2. Login
3. Change Password
4.. Delete User"
5. Display All Users (Testing)
6. TotalUsers
7. Update Usernames
8. Display Contents of a bucket
9.Clearing all the contents
10. Exit
